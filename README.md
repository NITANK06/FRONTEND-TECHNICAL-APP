# FRONTEND-TECHNICAL-APP


A modern admin dashboard built with Next.js, Material UI (MUI), Zustand, and DummyJSON API.

**This project demonstrates**

 Authentication using DummyJSON Auth API
 Protected routes
 Users management
 Products management
 Zustand state management
 Responsive UI with MUI
 Pagination, Search & Filters
 Client-side caching
 Performance optimization


# Tech Stack

 Next.js 15
 TypeScript
 Material UI (MUI)
 Zustand
 NextAuth.js
 Axios
 DummyJSON API



** Features**

** Authentication**

 Admin Login Page
 DummyJSON authentication API integration
 Protected dashboard routes
 Persistent login using localStorage

** Users Module**

 Users listing
 Search users
 Pagination
 Single user details page
 Responsive table/card UI

** Products Module**

 Products listing
 Product search
 Category filtering
 Pagination
 Single product details page
 Responsive product grid

**State Management**

**Managed using Zustand**

 Authentication state
 Users state
 Products state
 Async API actions

** Performance Optimization**

 React.memo
 useMemo
 useCallback
 API-side pagination
 Client-side caching

** UI/UX**

 Fully responsive design
 Material UI components
 Clean dashboard layout



** Why Zustand?**

Zustand IS chosen because

 Has a very small bundle size
 Requires minimal boilerplate
 Supports async actions directly
 Is simple and scalable
 Easier to manage than Redux for small-medium applications
